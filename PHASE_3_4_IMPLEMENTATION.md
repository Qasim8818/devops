**Q: Profile README not showing on my profile?**

**Q: My pinned repo isn't showing?**
A: Wait 5 minutes for GitHub to refresh. Then try refreshing your browser. Check that repo is PUBLIC.

**Q: Email not visible?**
A: Make sure you filled in the Email field in Settings → Profile tab. It must be a valid email.

**Q: Links aren't working?**
A: Check the markdown formatting in your README. Links should be `[text](url)` format.
